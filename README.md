### **Prerequisites**

```bash
# Debian/Ubuntu
sudo apt update && sudo apt install -y wget tar git make gcc binutils file xxd

sudo pacman -S wget tar git make gcc binutils file xxd

### **One-Command Setup & Run**

```bash
# Clone the repository
git clone https://github.com/SleepTheGod/PS3-RE.git
cd PS3-RE

# Make the script executable
chmod +x main.sh

# Run GOD MODE RE
./main.sh
